# My Journey

A timeline of my growth as a developer, from early experiments to launching production‑ready apps.

## 2026

- **Mar 2026** – Completely rewrote **PestiGuide** solo: rebuilt frontend with React + Tailwind, added JWT authentication, and made the entire platform responsive. Also launched this redesigned portfolio with MDX‑powered case studies, dark mode, and a focus on performance.

## 2025

- **Dec 2025** – Started internship at **MoonMVP** as a Full Stack Developer Intern, building end‑to‑end features with React, Node.js, and databases.
- **Nov 2025** – Built **ThugTap**, a multiplayer Ludo platform with wallet system, admin panel, and automated payouts – delivered to a client in 15 days.
- **Aug 2025** – Created my first simple portfolio website to showcase early projects.
- **Jun 2025** – Completed **Rentpy**, a full‑stack rental marketplace with JWT auth, Google Maps, and Razorpay payments (solo, 3‑month build from March to June).
- **May 2025** – Started working on **PestiGuide** as a team project (backend lead); began exploring MongoDB, Express, and collaborative Git workflows.

## 2024

- **Dec 2024** – Deepened full‑stack skills with React, Node.js, Express, and databases (PostgreSQL, MongoDB). Worked on several mini‑projects and contributed to open‑source.
- **Oct 2024** – Began serious full‑stack development, building APIs and integrating frontend with backends.
- **Jan – Sep 2024** – Continued learning: advanced React patterns, Tailwind CSS, Node.js, and database design.

## 2023

- **Oct 2023** – Started coding seriously, learning HTML/CSS, JavaScript, and Git. Built first small interactive pages.
- **Aug 2022 – Sep 2023** – Began computer science engineering at Medi‑Caps Institute; explored programming fundamentals.
