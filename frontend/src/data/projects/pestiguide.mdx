# PestiGuide: Pesticide Information Platform

## Overview

PestiGuide is a full‑stack platform that helps farmers access accurate, crop‑specific pesticide information. Originally a team project, I later took full ownership, completely rebuilt the frontend with React and Tailwind CSS, and enhanced the backend with user authentication. The result is a production‑ready application that combines a clean, responsive interface with powerful data capabilities.

## What Users Can Do

- **Browse by Season** – View crops grouped by Kharif, Rabi, or Zaid seasons.
- **Search & Filter** – Quickly find crops or pesticides via a smart search bar.
- **Crop‑Specific Pesticides** – See all recommended pesticides for a chosen crop, with detailed profiles.
- **Pesticide Details** – Access usage instructions, active ingredients, application methods, environmental impact, advantages, and disadvantages.
- **Authentication** – Sign up / log in to access full content (pesticide details are protected).
- **Dark Mode** – Toggle between light and dark themes for comfortable viewing.

## Why I Built This

Farmers often struggle to find reliable, crop‑specific pesticide information. Local advice can be inconsistent or inaccurate. PestiGuide provides a trusted, data‑driven reference that is easy to navigate and always up‑to‑date. I built the entire application to deliver a seamless user experience while demonstrating my full‑stack skills.

## Tech Stack

- **Frontend**: React, Tailwind CSS, Axios, React Router
- **Backend**: Node.js, Express.js, MongoDB, JWT authentication
- **Deployment**: Vercel 

## Technical Highlights

### Full‑Stack Architecture

- **React frontend** with component‑based design and responsive Tailwind CSS.
- **Node.js/Express backend** providing RESTful APIs.
- **MongoDB database** storing crops, pesticides, and their relationships.

### Authentication & Security

- JWT‑based authentication with tokens stored in `localStorage`.
- Protected routes: pesticide details are only accessible to logged‑in users.
- User signup and login with validation and error handling.

### Data Management

- MongoDB aggregation pipelines to fetch crop‑specific pesticides efficiently.
- Schema design that captures seasonal crops, pesticide details, and many‑to‑many relationships.

### UI/UX

- Intuitive home page with a hero section, search bar, and season selection buttons.
- Dedicated pages for season‑based crops, crop‑specific pesticides, and pesticide details.
- Responsive design that works on mobile, tablet, and desktop.
- Dark mode toggle for comfortable viewing at any time of day.

## Future Plans

- Add user‑generated reviews and ratings for pesticides.
- Implement a “favorites” system so users can save pesticides.
- Expand the database with more crops and pesticides.
- Deploy the application to a live domain for public access.
